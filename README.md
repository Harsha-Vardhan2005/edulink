# EduLink ‚Äî The Ultimate Social Learning Platform

EduLink is a modern **social media platform for learners**. It blends traditional social networking with educational roadmaps, allowing students to connect, share, and grow together while learning new skills such as **Machine Learning**, **Deep Learning**, and more.

---

## Features

### Authentication
- **Login and Signup** using **Google Firebase Authentication**
- Secure and user-friendly access for all users

###  Social Interaction
- Create and share **posts**
- Add **likes** and **comments**
- Participate in **polls** and discussions

### Connect & Collaborate
- **Follow** and **connect** with other learners  
- Once connected, users can:
  - **Send messages**
  - **Make audio/video calls** using **WebRTC**

### Learning Roadmaps
- Integrated **Roadmap Feature** powered by **Gemini API**
- Personalized paths for learning skills like **ML**, **DL**, and more

### Random Call Feature
- Engage with the community through **randomized video calls**
- Designed to encourage new connections and collaborations
